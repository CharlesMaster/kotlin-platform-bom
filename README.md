Kotlin Platform BOM
===================

A BOM for Kotlin platform

### How to use ?

Please add following code in your pom.xml:

```xml
<dependencyManagement>
     <dependencies>
         <dependency>
             <groupId>com.github.linux-china</groupId>
             <artifactId>kotlin-platform-bom</artifactId>
             <version>1.0.0-SNAPSHOT</version>
             <type>pom</type>
             <scope>import</scope>
         </dependency>
     </dependencies>
 </dependencyManagement>
```
